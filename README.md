### Requirements
- A Minecraft account
- A Discord bot:

    - `Manage Webhooks`
    - `View Channels`
    - `Add Reactions`
    - `Send Mesaages`

    - `bot`

- **Error**: Errors
- **Warn**: Warnings
- **Info**: General information
- **Debug**: Specific information
- **Trace**: All the information
